# 📊 Frog Jump Web Leaderboard

![Vercel](https://img.shields.io/badge/Deployment-Vercel-black.svg)
![JS](https://img.shields.io/badge/Language-JavaScript-F7DF1E.svg)

Frog Jump Game의 전역 순위를 웹 브라우저에서 실시간으로 확인할 수 있는 대시보드입니다.

## 🌟 주요 특징
- **실시간 자동 갱신**: 5초 간격으로 서버에서 최신 랭킹 데이터를 가져와 자동으로 화면을 업데이트합니다.
- **순위 하이라이트**: 상위 1, 2, 3위 사용자에게는 특별한 배경색(금, 은, 동)을 부여하여 시각적 효과를 극대화했습니다.
- **반응형 디자인**: PC와 모바일 환경 어디서나 깔끔하게 순위표를 확인할 수 있습니다.

## 🔗 접속 주소
- **URL**: [https://frogjump-leaderboard-web.vercel.app/](https://frogjump-leaderboard-web.vercel.app/)

## 🛠 기술 스택
- **Frontend**: HTML5, Vanilla CSS, JavaScript (Fetch API)
- **API Server**: Render (FastAPI)
